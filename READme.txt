# Коментарии
--------------------------------------------------------------------------------------------	ШПОРА
cd Projekt/PandemicProject/	Это выполняется при запуске (переход в каталог проекта)
git pull			Сразу обновляем прошлые изменения из сервера
git log				Показать историю изменений (можно понять, 
				были ли изменения, и если да, то какие)
--------------------------------------------------------------------------------------------
git add Step1/   		После редактирования добавляем все изменения активного
				каталога в облласть коммита

git commit -m "Коментарий"	Добавляем промежуточную версию (коммит)

git push 			Отправить локальный репозиторий на сервер
				(можно в конце)

git status			Показать статус текущих коммитов
